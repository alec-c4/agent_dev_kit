# Kit agents (Claude Code)

Optional **persona prompts** for Claude Code subagents. Installed to `~/.claude/agents/` via `./scripts/kit install --target=claude`.

| Agent | Use when |
|-------|----------|
| [developer.md](developer.md) | Implement from approved spec and plan |
| [architect.md](architect.md) | Design, tradeoffs, ADR-level decisions |
| [auditor.md](auditor.md) | Security and compliance review |
| [explore.md](explore.md) | Read-only codebase exploration |
| [orchestrator.md](orchestrator.md) | Multi-step workflow coordination |

All agents defer to [AGENTS.md](../AGENTS.md) and `docs/guidelines/` — no duplicate universal rules.
